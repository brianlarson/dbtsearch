# Legacy page markup (reference)

**Summary:** Static HTML snapshots of the legacy React app live in `docs/reference-markup/` for Bootstrap → Tailwind parity work. Asset paths in those files point at `../../public/` so CSS and images work when you open a file from disk.

The capture scripts and root React/Vite app were removed from this branch. To re-capture markup, check out `main` or `legacy`, run the old client, and save pages into `docs/reference-markup/` (or restore the capture scripts from git history).

**Output already present:** `docs/reference-markup/home.html`, `providers.html`, `about.html`, `faqs.html`, `contact.html`, `register.html`, `login.html`, `logout.html`, `404.html`, plus `README.md`.
