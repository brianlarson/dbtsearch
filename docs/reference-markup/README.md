# Legacy page markup (TT-14)

Rendered HTML of each legacy route for Bootstrap → Tailwind + Vue reference.

**Capture:** On `main`, set `VITE_CAPTURE_MARKUP=true`, run `npm run client`, then in another terminal run `npm run capture-markup`. If the client runs on a different port (e.g. 5175), run `npm run capture-markup -- http://localhost:5175`.

If files are ~2KB each (shell only), re-run with the client’s URL so the SPA has time to render.
