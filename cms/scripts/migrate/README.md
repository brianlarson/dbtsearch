# Local migration workspace (gitignored)

This folder is intentionally **gitignored** so you can keep local-only migration helpers here
without committing them to the repo.

## How to use

- Put any local scripts, configs, scratch notes, etc. in this directory.
- Everything is ignored by git **except** `.gitignore` and this `README.md`.

## Notes

- If you need to share a migration script with the team, place it somewhere else in the repo
  (or change `cms/scripts/migrate/.gitignore` to explicitly allow it).
